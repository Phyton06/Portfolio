## Resumen

SynthAI es un asistente de code review impulsado por modelos de lenguaje. Analiza cada PR en busca de vulnerabilidades de seguridad, code smells y oportunidades de optimización.

## Arquitectura

- **Frontend**: React con Vite, integración vía GitHub App
- **Backend**: Python (FastAPI) con workers asíncronos
- **IA**: Modelos fine-tuned sobre CodeLlama desplegados en AWS SageMaker
