## Resumen

NeoDeploy es un orquestador de despliegues diseñado para equipos que necesitan releases frecuentes sin downtime. Soporta estrategias blue-green, canary y rolling updates.

## Arquitectura

- **Motor de despliegues**: Go con workers concurrentes
- **Orquestación**: Kubernetes CRDs para definiciones de deploy
- **Cache y estado**: Redis con persistencia
- **CI/CD**: Integración nativa con GitHub Actions
