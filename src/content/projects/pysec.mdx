## Resumen

PySec Audit es una herramienta de seguridad especializada en ecosistemas Python. Escanea dependencias en busca de vulnerabilidades conocidas, genera reportes SBOM y bloquea pipelines inseguros.

## Arquitectura

- **Scanner**: Python con base de datos de vulnerabilidades actualizada
- **Reportes**: Generación de SBOM en formato SPDX y CycloneDX
- **Integración**: GitHub Actions, GitLab CI, Jenkins
- **Infraestructura**: AWS Lambda + ECR para escaneo serverless
