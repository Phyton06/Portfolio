## Resumen

TaskFlow es una plataforma de gestión de proyectos que combina tableros colaborativos en tiempo real con automatización inteligente. Diseñada para equipos que necesitan visibilidad instantánea del flujo de trabajo.

## Arquitectura

- **Frontend**: React con TypeScript, estado global con Zustand
- **Backend**: Python (FastAPI) con WebSockets para actualizaciones en vivo
- **Infraestructura**: Docker Compose con Nginx como reverse proxy
