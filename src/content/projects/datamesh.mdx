## Resumen

DataMesh es una plataforma de procesamiento de datos en tiempo real construida sobre una arquitectura event-driven. Procesa millones de eventos por minuto con latencia sub-segundo.

## Arquitectura

- **Procesamiento**: Go con Apache Kafka para streaming
- **Almacenamiento**: AWS S3 + Redshift para data lake
- **Visualización**: Grafana con dashboards dinámicos
